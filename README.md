# PotPlayer 迅雷字幕搜索插件

这是一个专为 **PotPlayer** 开发的在线字幕搜索插件，通过调用迅雷字幕 API 获取并加载高质量的在线字幕。

## 📥 安装说明

### 1. 获取插件文件
* 下载本项目的所有文件

### 2. 放入 PotPlayer 插件目录
1. 打开 PotPlayer 的安装目录（通常在 `C:\Program Files\DAUM\PotPlayer\Extension\Subtitle\Search` 或 `C:\Program Files (x86)\DAUM\PotPlayer\Extension\Subtitle\Search`）。
    * *注：如果没有文件夹，可以手动创建对应名称的文件夹。*
2. 将插件文件放入该路径下。

### 3. 在播放器中启用
1. 打开 PotPlayer。
2. 播放任意视频后，右键点击播放器界面，选择 **字幕** -> **在线字幕搜索**。
3. 在弹出的搜索界面或下拉菜单中，应该能够看到 **“迅雷在线字幕”** 的选项。

---

## 🚀 使用说明

1. **自动搜索：** 播放视频时，右键选择 `字幕` -> `在线字幕搜索`，插件会自动读取当前文件的元数据或文件名进行查询。
2. **手动查询：** 如果自动搜索结果不匹配，可以在搜索窗口中输入您想要的电影/剧集名称进行手动检索。
3. **下载应用：** 找到合适的字幕后，直接双击或点击下载，播放器将自动加载并应用该字幕。
